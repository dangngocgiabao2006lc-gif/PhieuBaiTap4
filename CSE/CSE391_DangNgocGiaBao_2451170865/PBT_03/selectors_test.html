Giá sản phẩm

Và các CSS rules: p { color: black; } /* Rule A */ .price { color: blue; } /* Rule B */ #main-price { color: red; } /* Rule C */ p.price { color: green; } /* Rule D */ 1. Tính specificity score Rule Selector Score A p (0,0,1) B .price (0,1,0) C #main-price (1,0,0) D p.price (0,1,1) 2. Element sẽ có màu gì? Giải thích Kết quả red Giải thích Rule C: #main-price { color: red; } có ID selector nên specificity cao nhất. So sánh: (1,0,0) > (0,1,1) nên browser chọn màu đỏ. 3. Nếu thêm inline style

Kết quả orange Giải thích Inline CSS có độ ưu tiên cao hơn: ID Class Element selector 4. Nếu Rule A thêm !important p { color: black !important; } Kết quả black Giải thích !important override specificity thông thường nên: color: black !important; sẽ thắng cả rule có ID selector.
